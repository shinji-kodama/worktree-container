# Security Policy

## Reporting Vulnerabilities

If you discover a security issue, **please do not create a public Issue**.

Instead, please report it privately through [GitHub Security Advisory](https://github.com/mmr-tortoise/loam/security/advisories/new).

### Information to Include in Your Report

- Description of the vulnerability
- Steps to reproduce
- Assessment of the impact scope
- A proposed fix, if possible

### Response Process

- We will acknowledge your report within **48 hours**
- After a fix is prepared, it will be published as a security advisory

## Supported Versions

Security updates are provided only for the latest major version.
